PEEKY's Commands


Function Commands
Command Name Description Usage Permissions Needed
Overview Gives you the list of all the functions and their current state and setting. p!overview None
Toggle Toggles a function's state. p!toggle [function name] MANAGE_GUILD
Set Sets a function's specific setting, if available. p!set [function name] [setting] MANAGE_GUILD


Moderation Commands
Command Name Description Usage Permissions Needed
Mute Mutes the mentioned user. p!mute [user mention] [reason] MUTE_MEMBERS
Unmute Unmutes the mentioned user. p!mute [user mention] [reason] MUTE_MEMBERS
Purge Purges a specific amont of messages. p!purge [number] MANAGE_MESSAGES
IDBan Bans a user outside the server based on their ID. p!idban [user id] BAN_MEMBERS
Ban Bans a user inside the server. p!idban [user mention] [reason] BAN_MEMBERS
Lockdown Locks the current channel for a specific role. This can only be reverted manually as it's supposed to be used in an emergency. p!lockdown [role mention] MANAGE_CHANNELS
MutedRole Changes the name of the Muted role that PEEKY uses for the moderation commands and some functions. p!mutedrole [role name] MANAGE_GUILD
Prefix Changes the prefix for the entire server. The prefix for this command MUST be the default one. p!prefix [text] Server Owner


Music Commands
Command Name Description Usage Permissions Needed
Play Plays a song from the given YouTube URL. p!play [YouTube URL / "random"] None
Current Shows what song is currently being played and how much time until it ends. p!current None
Leave Stops the music and makes PEEKY leave the Voice Channel. p!leave None


Profile Commands
Command Name Description Usage Permissions Needed
Profile Shows a profile of you or the person you've mentioned. p!profile [user mention] None
Inventory Shows someone's very own inventory that has their backgrounds and badges. p!inventory [user mention] None
BuyBackground Buys the specified background for your profile and adds it your inventory for a specific price. p!buybackground [background ID] None
SetBackground Sets the specified background for your profile, but only if you already have it in your inventory. p!setbackground [background ID] None
SellBackground Sells a background from your inventory for 50 Gredit. p!sellbackground [background ID / "all"] None
BuyDescription Buys a description for your profile at the price of 25 Gredit. p!buydescription [text / "random"] None
Daily Gives you a daily reward. p!daily None
Redeem Rewards you for items bought from the store. p!redeem None
Gift Give someone some of your items like backgrounds, chests, etc... p!gift [user mention] ["gredit" / "chest" / "background"] [number] None
Open Opens one or more of your chests for a nice loot. p!open [nothing / "all"] None


Miscellaneous Commands
Command Name Description Usage Permissions Needed
Help Shows PEEKY's help menu. p!help None
SetInvite Searches the current channel for an server invite to use in the server list. (Upgraded Servers only) p!setinvite Server Owner
Get Transforms provided text like roles, emojis, mentions to get their IDs. p!get [text] None
Eval Runs the provided code. p!eval [code] PEEKY's Owner
