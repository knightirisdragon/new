Ambassador Program The Ambassador Program is a way for you to get rewarded just for telling your friends about PEEKY!

The way it works is that you create your very own invite link in the #home channel of the Support Server and then send it to your friends.
After your friend joins the server using your invite link, you should get a message from the bot saying that it worked!


These are all the reward

Chest
Awarded when someone uses your invite link.

Ambassador Badge
Awarded when someone uses your invite link for the 1st time.

Exclusive "Envelope" background
Awarded when someone uses your invite link for the 5th time.

Server Upgrade
Awarded when someone uses your invite link for the 10th time.

Premium (30 days)
Awarded when someone uses your invite link for the 25th time.
