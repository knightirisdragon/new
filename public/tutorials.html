Starting with PEEKY
1. How to use PEEKY?
You can start using PEEKY the big way by getting familiar with the commands.

2. How to use the functions?
Setting up PEEKY's functions is super easy once you get the hang of it!

After you've got familiar with the commands, you know that you only need to use Set and Toggle to change a function's behaviour.
You can use both commands by typing the command and then the function's Command Name, which you can find by in the Functions page.

The Toggle command toggles between the function's state, which can be on or off.
It can also create channels and roles under the correct circumstances (which means just having the default permissions).

The Set command on the other hand changes a function's setting, which differs with each function.
You can see what the settings do for each functions in the Overview command.

There's also a Clear command that basically clears the setting of a function that uses arrays as it's setting.
For example; the Banned Words function has an array of banned words that can be cleared using this command.

3. What are the functions types?
All the functions are arranged into two types: Channel type and Server type.
You can see the type of a function in the Overview command.

The Server type means that the function is always gonna be enabled or disabled for the whole server
and the Channel type means that the function can be enabled or disabled in every channel in the server.

The Server Message function and it's Tags
1. What are Tags?
The Tags in the Server Message function are a great thing that let you customize your server message even more!
In this tutorial you'll learn how to use them what they each tag can do.

2. How to use Tags?
You can use the tags just by typing them between square brackets like [This] in the function's setting.
You'll know if it worked by getting a "Tag was detected" message right after setting the Server Message function.

3. What Tags are there?
These are the Tags that you can currently use:

[GuildName] - Shows the server's current name.
[GuildSize] - Shows the amount of members in the server. Excludes bots.
[GuildOwner] - Shows server's current owner.
[GuildLevel] - Shows the current verification level.
[GuildAcronym] - Shows the server's Acronym.

About the Workshop and backgrounds
1. Submitting a background
The parameters that PEEKY uses for the profiles is 500x300.

This is the format that you should use when submitting a background to the Workshop:

Name: [text]
Credit: [text]
Price: [number]

Having an invalid format or not attaching your background will get your submission automatically deleted.
Posting something NSFW, racist, or anything of that sort will get you banned from submitting to the Workshop.

2. Background Revenue?
Background revenue means that the background's artist gets some Gredit from the background's sales.
You can see if a background has revenue enabled when it says "Revenue Enabled" under it's price tag.

To get revenue enabled on your background, keep in mind that:
- You must be the background's original author or have a special permission from the original artist.
- Your profile must be active, otherwise the revenue will be turned off until you get one.

3. Exclusive backgrounds?
Only PEEKY's developers can add Exclusive backgrounds to the Workshop.

Exclusive backgrounds are backgrounds that cannot be bought with Gredit, but instead have to be earned through special events and so on.
Selling an exclusive background will give you no Gredit.

Some tips about the Store
1. How do I get my items?
You should normally get a DM from PEEKY saying that the items have been added to your profile right after you purchase said items.
If you don't receive the DM, try waiting a few hours or restart your client.

2. How do I get a refund?
The steps on how to issue a refund are in the Refund Policy page.

3. Re-purchasing the Supporter item
Once you purchase the Supporter item, you will hold the Supporter status for 30 days.
After the 30 days, PEEKY will remove the status and notify you, after which you have 5 days to re-purchase it in order to not lose the date of the initial purchase.

About the Auto Wipe system
1. What is this?
Auto Wipe is a system that helps PEEKY by removing inactive or unavailable servers and profiles.

2. Normal Servers
PEEKY will leave a server and delete it's settings if it has been inactive for more than 20 days.
Servers can update their Auto Wipe status by using PEEKY's functions and commands.

3. Upgraded Servers
Upgraded servers are completely immune to Auto Wipe.

4. Profiles
Any profile that are not available to PEEKY or don't have the Fashion badge will get deleted after 20 days of inactivity.
Profiles can update their Auto Wipe status by using PEEKY's commands.

Help with the Server List
1. How do I get my server on the list?
The server list is a reward for anyone that has upgraded their servers.
Go to the store to upgrade your server!

2. How do I update my server's invite link?
To update your invite link in the server list, you have to make a permanent invite and then use the SetInvite command.

3. My upgraded server is not on the list?
It takes some time for the list to update.
If your server still is not on the list, it may be because PEEKY is not on the upgraded server.
